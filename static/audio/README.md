# Audio Files

Placez vos fichiers audio ici.

## Fichier requis

- `background-music.mp3` - Musique de fond pour la condition "music"

## Format recommandé

- Format : MP3
- Bitrate : 128-192 kbps
- Durée : Au moins 20 minutes (ou en boucle)
- Volume : Normalisé à -6dB

## Exemple

```
static/audio/
└── background-music.mp3
```

## Note

L'application bouclera automatiquement le fichier audio pendant toute l'expérience.
